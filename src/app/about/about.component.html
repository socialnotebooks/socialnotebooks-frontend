About Social Notebooks

Behind the Scenes: Our Tech Stack

The magic of Social Notebooks lies in its seamless user experience, powered by a modern and robust technology foundation:


Front End:

Built with Angular, HTML, and CSS, our interface is designed for a smooth and responsive experience.

Back End:

Our core services are built on .NET Core and hosted on Azure App Service, delivering reliable performance and scalability.

Data & Storage:

We securely manage your content using Azure Cosmos DB and Blob Storage.

Security & Optimization:

Azure Key Vault protects sensitive data.

One-Way Encryption:

All messages and calls are secured using one-way encryption, ensuring that your communications remain confidential.

Azure Cognitive Search:

Leverages AI to provide powerful, context-aware search capabilities.

API Management & Connectivity:

Azure API Management (APIM): Ensures that our APIs are secure, scalable, and easy to integrate.

Peering Between VMs & App Services:

Enhances performance and security through efficient, direct communication between our hosting environments.

Open Source at Our Core:

We embrace open source principles, ensuring transparency, flexibility, and community-driven innovation.
