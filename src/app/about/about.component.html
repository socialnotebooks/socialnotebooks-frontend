About Social Notebooks

Welcome to Social Notebooks — a thoughtfully designed platform for sharing your voice, creativity, and everyday moments with a focus on simplicity and user privacy.

🌍 Our Vision

We created Social Notebooks to bring back authenticity in online sharing. Unlike mainstream platforms, we prioritize content over clout — giving you a quiet, elegant space to post freely and discover organically.

Whether you're posting art, thoughts, or moments, Social Notebooks is your digital canvas.


🛠 Tech Stack

Social Notebooks is powered by a modern, scalable, and secure technology stack:


Front End:

Angular 12 — for a dynamic, component-driven UI

Back End:

ASP.NET Core Web API — for fast, secure, and scalable APIs

Database:

Azure SQL Database — highly available and optimized for performance

Cloud Hosting:

Microsoft Azure — enterprise-grade cloud infrastructure

File Storage:

Azure Blob Storage — for efficient and secure media hosting

Authentication & Security:

Azure Active Directory & HTTPS — ensuring data privacy and secure access

Real-time Updates:

Azure Service Bus — delivering new feeds to users in real-time
Our entire stack is cloud-native and built with scalability, privacy, and performance in mind.
